# Tasks — SPEC-MOUSE-001

TDD (RED → GREEN → REFACTOR) 사이클에 따라 4개의 수직 슬라이스를 원자적 태스크로 분해한다. 각 태스크는 하나의 TDD 사이클로 완료 가능하며 (≤ ~150 LOC 테스트 포함), `Status` 는 전부 `pending` 으로 시작한다.

**슬라이스 순서 (엄격 준수):**
1. Slice 1 — transport + protocol (T-001 ~ T-009)
2. Slice 2 — ownership FSM + edge detector + takeback (T-010 ~ T-019)
3. Slice 3 — input capture/inject + visibility (T-020 ~ T-029)
4. Slice 4 — host/remote/cli orchestration + config + E2E (T-030 ~ T-036)

각 슬라이스는 직전 슬라이스의 모든 태스크가 녹색(coverage ≥ 85%)이 된 이후에만 착수한다.
